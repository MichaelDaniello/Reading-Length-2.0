# Reading Length
Repository for the redesign of Reading Length

## Installation
> yarn install

### Local development
> yarn run dev

### Dev deployment
> yarn run build && cd build && now

### Production deployment
Create a pull request to the master branch

## Technologies used
### Front-end
* React
* NextJS
* GraphQL
### Back-end
* Apollo
* PassportJS
* GraphQL
### Database
* Prisma
* MongoDB
